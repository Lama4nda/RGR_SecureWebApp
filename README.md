**SecureWebApp - Менеджер Відпочинку**

Це RESTful веб-застосунок для управління курортами, розроблений в рамках розрахунково-графічної роботи. 
Проект реалізований на Java з використанням Spring Boot та розмежування прав доступу. Проект має простий веб інтерфейс і можливість авторизації користувачів.
Система дозволяє керувати базою даних туристичних напрямків, включаючи ієрархічну структуру: *Країни* → *Курорти* → *Готелі.* Є можливисть бронювання номеру в готелі та вибору чим зайнятися на курорті. Якщо країна чи курорт зачинений бронювання недоступне.

**Налаштування та Запуск**

1.*Запуск на сервері (база даних + застосунок):*
```bash
docker-compose up -d --build
```
2.*Запуск на сервері тільки бази даних (Застосунок запускається в IDE):*
```bash
docker-compose up -d db
```
**Доступ (Логін/Пароль)**
При першому запуску автоматично створюються наступні користувачі:

| Роль | Логін | Пароль | Права |
| --- | --- | --- | --- |
| Admin | admin | admin | Має повний доступ (Create, Read, Update, Delete) |
| User| user| user | Може переглядати дані та бронювати номер в готелі |
